HTML je osnovni jezik koji se koristi za kreiranje i strukturiranje sadržaja na veb stranicama. Skraćenica "HTML" označava "Hypertext Markup Language". HTML koristi "oznake" (tags) kako bi definisao različite elemente na veb stranici, poput naslova, paragrafa, slika, veza i mnogih drugih. Ove oznake pružaju strukturu i semantičko značenje sadržaja, omogućavajući pretraživačima i drugim uređajima da bolje razumeju stranicu.

CSS je jezik za stilizovanje i oblikovanje veb stranica. Skraćenica "CSS" označava "Cascading Style Sheets". Dok HTML definiše strukturu sadržaja, CSS se koristi za stilizovanje tog sadržaja, čineći ga vizualno privlačnim i uređenim. CSS omogućava podešavanje boja, fontova, rasporeda, veličine i drugih vizuelnih svojstava elemenata na stranici.
